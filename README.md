# Proyecto de Clasificación de Especies de Iris

## Propósito
Este proyecto clasifica especies de flores Iris (Setosa, Versicolor, Virginica) usando el dataset de Iris con 4 características: longitud y ancho del sépalo y pétalo. Aplicamos un pipeline completo de minería de datos y presentamos resultados en un dashboard interactivo con Streamlit.

## Instrucciones
1. Clona este repositorio.
2. Instala dependencias: `pip install -r requirements.txt`.
3. Ejecuta `python model_training.py` para entrenar el modelo y generar métricas.
4. Ejecuta `streamlit run proyect.py` para lanzar el dashboard.
5. Para la presentación en video, explica el workflow, elecciones (ej. Random Forest por robustez) y demuestra el dashboard.

## Workflow
- Entendimiento de Datos: Exploramos distribuciones y correlaciones.
- Preprocesamiento: Estandarizamos características.
- Modelado: Clasificador Random Forest.
- Evaluación: Precisión, Precision, Recall, F1 y validación cruzada.
- Dashboard: Predicciones interactivas y visualizaciones.
